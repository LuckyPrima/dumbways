# Personal Portfolio – Lucky Prima Putra Helen

Portfolio web pribadi yang dibangun menggunakan **HTML** dan **Tailwind CSS**. Website ini menampilkan informasi singkat mengenai profil, skill, pengalaman, serta proyek yang pernah dikerjakan. Desain dibuat sederhana, bersih, responsif, dan mudah dikembangkan.

## 🚀 Features

- **Profile Section**
  Menampilkan foto profil, bio singkat, asal daerah, dan tautan sosial media (LinkedIn, Instagram, GitHub).

- **Responsive Layout**
  Tampilan fleksibel dan rapi pada perangkat mobile maupun desktop.

- **Modern Card UI**
  Setiap section menggunakan card dengan:

  - Rounded corners
  - Soft shadow
  - Hover zoom animation

- **Social Media Icons**
  Menggunakan ikon SVG berwarna putih dengan background biru muda.

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CDN)
- SVG Icons
- Responsive Layout (Flexbox & Grid)

## 📁 Project Structure

```
/
├── index.html        # Halaman utama portfolio
├── public
    ├── profile.jpg   # Foto profil (opsional)
└── README.md         # Dokumentasi
```

## ⚙️ How to Use

1. Clone repository:

```
git clone https://github.com/LuckyPrima/dumbways_task_1.git
```

2. Letakkan file `profile.jpg` di public folder.

3. Buka langsung file berikut di browser:

```
index.html
```

Tidak memerlukan server atau proses build.

## 🎨 Customization

Anda dapat menyesuaikan:

- Warna tema
- Animasi hover
- Section tambahan (Contact, Education, Tech Stack)
- Ikon sosial media

Tailwind CSS memungkinkan perubahan yang cepat dan fleksibel.
